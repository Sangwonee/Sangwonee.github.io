> 안녕하세요, 기술을 통해 아이디어를 현실로 만드는 개발자 윤상원입니다.

## 주요 관심 분야

저는 특정 분야에 얽매이지 않고, 웹 개발부터 클라우드 인프라(AWS), 그리고 인공지능(AI)에 이르기까지 폭넓은 기술 스펙트럼에 걸쳐 꾸준히 성장하고 있습니다. 빠르게 변화하는 기술 트렌드 속에서 문제 해결을 위한 최적의 도구를 선택하고 활용하는 능력을 기르고자 합니다.

*   **Web Development**: 견고하고 확장성 있는 웹 애플리케이션을 만듭니다.
*   **Cloud Infrastructure (AWS)**: 안정적인 서비스 운영을 위한 인프라를 설계하고 관리합니다.
*   **Artificial Intelligence (AI)**: 데이터를 통해 새로운 가치를 창출하는 기술에 주목하고 있습니다.

## 핵심 역량

- **Backend Development (Java/Spring)**: Spring Boot, Security, JPA를 사용하여 인증/인가 기능이 포함된 RESTful API 서버를 개발합니다. MySQL 데이터베이스 연동 경험이 있으며, 빌드 도구로 Gradle을 사용합니다.

- **Containerization**: Docker를 사용하여 애플리케이션을 컨테이너 이미지로 빌드하고, 일관성 있는 개발 및 배포 환경을 구축하는 역량을 갖추고 있습니다.

- **Deployment & Operation (AWS)**: GitHub Actions 등 CI/CD 도구를 활용해 배포 자동화 파이프라인을 구축합니다. AWS의 주요 서비스(EC2, S3, RDS)를 기반으로 로드 밸런서와 오토 스케일링 그룹을 구성하여 트래픽을 효율적으로 분산하고, CloudWatch 알람과 연계하여 장애 상황에 자동화된 대응이 가능합니다.

- **Data Science (Python)**: Pandas, Scikit-learn 등 Python 라이브러리를 활용한 데이터 분석 및 머신러닝 모델 개발 역량을 갖추고 있습니다.
